# Daily Report Template

## HTK X Outreach Daily Report

**VA:** Record your full name in the submitted report.  
**Date:** Record the work date in `YYYY-MM-DD` format.  
**Shift:** Record the start and end time with time zone.  

## Daily KPI Summary

| KPI | Target | Actual |
| --- | ---: | ---: |
| New personalized DMs sent | 20 | Record today's count |
| Follow-ups sent | 15 | Record today's count |
| Conversations started | 6 | Record today's count |
| Qualified leads | 2 | Record today's count |
| Calls scheduled | 1 | Record today's count |
| Opt-outs | 0 expected | Record today's count |
| Account warnings or limitations | 0 expected | Record today's count |

## Scheduled Calls

For each call scheduled today, report:

- Prospect name
- X handle
- Call date, time, and time zone
- Prospect segment
- Performance goal
- Main obstacle
- Why now
- Qualification score

Write `No calls scheduled today` when the list is empty.

## Best Conversations

Report the three most promising active conversations:

1. Prospect X handle, current stage, goal, obstacle, and next action.
2. Prospect X handle, current stage, goal, obstacle, and next action.
3. Prospect X handle, current stage, goal, obstacle, and next action.

If fewer than three conversations are active, report every active conversation and write `No additional active conversation`.

## Follow-Up Queue

Report:

- Number of follow-ups due tomorrow
- Number of active conversations awaiting a prospect reply
- Number of qualified leads waiting for scheduling details
- Number of scheduled calls occurring in the next 48 hours

## Blockers And Escalations

Report:

- Brand-risk conversations
- Medical or injury questions
- Customer-service issues
- Partnership or media requests
- Public-figure or organization accounts
- Account-security issues
- X warnings or limitations

Write `No blockers or escalations today` when the list is empty.

## Quality Check

Answer `Yes` or `No`:

- Did every first-contact DM use a real profile signal?
- Did you avoid sending a link in every first-contact DM?
- Did you stop every sequence after an opt-out or rejection?
- Did you update every touched CRM record?
- Did you avoid unapproved tools, automation, and bulk sending?

If any answer is `No`, explain exactly what happened and alert the manager immediately.

## Daily Notes

Record:

- What message pattern created the best replies?
- What prospect segment responded best?
- What recurring objection appeared?
- What should the manager review?

## Submission Standard

Submit the report at the end of every shift. The daily report does not replace CRM updates.
