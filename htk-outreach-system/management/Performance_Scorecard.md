# VA Performance Scorecard

## Purpose

Use this scorecard weekly. The score does not replace judgment. Any serious compliance violation triggers immediate review regardless of total points.

## Scoring

| Category | Weight | Full-credit standard |
| --- | ---: | --- |
| Prospect quality | 20 | At least 85% of sampled prospects clearly fit HTK |
| Message quality | 20 | At least 90% of sampled first-contact DMs pass QA |
| Qualification quality | 15 | Qualified leads meet all five gates and calls are appropriate |
| Calls scheduled | 15 | At least 3 qualified calls scheduled in the week |
| CRM accuracy | 15 | At least 98% of required sampled fields complete |
| Responsiveness | 10 | Active conversations handled within 60 minutes during shift |
| Compliance | 5 | Zero violations, warnings ignored, or opt-out failures |

## Point Rules

### Prospect Quality: 20 Points

| Result | Points |
| --- | ---: |
| 90%-100% sampled prospects fit | 20 |
| 85%-89% fit | 17 |
| 75%-84% fit | 10 |
| Below 75% fit | 0 |

### Message Quality: 20 Points

| Result | Points |
| --- | ---: |
| 95%-100% sampled DMs pass | 20 |
| 90%-94% pass | 17 |
| 80%-89% pass | 10 |
| Below 80% pass | 0 |

### Qualification Quality: 15 Points

| Result | Points |
| --- | ---: |
| Every sampled qualified lead meets all five gates | 15 |
| One sampled lead has a minor documentation gap | 10 |
| One sampled lead was booked without a required gate | 5 |
| Multiple poor-fit calls were booked | 0 |

### Calls Scheduled: 15 Points

| Result | Points |
| --- | ---: |
| 5 or more calls | 15 |
| 4 calls | 13 |
| 3 calls | 10 |
| 2 calls | 6 |
| 0-1 calls | 0 |

### CRM Accuracy: 15 Points

| Result | Points |
| --- | ---: |
| 100% sampled fields complete | 15 |
| 98%-99% complete | 13 |
| 95%-97% complete | 8 |
| Below 95% complete | 0 |

### Responsiveness: 10 Points

| Result | Points |
| --- | ---: |
| At least 95% of shift-time replies handled within 60 minutes | 10 |
| 85%-94% handled within 60 minutes | 7 |
| 70%-84% handled within 60 minutes | 4 |
| Below 70% handled within 60 minutes | 0 |

### Compliance: 5 Points

| Result | Points |
| --- | ---: |
| Zero issues | 5 |
| One minor issue identified and corrected before sending | 3 |
| Any sent-message violation, ignored opt-out, automation use, or account-security issue | 0 and immediate review |

## Score Interpretation

| Score | Result | Management action |
| ---: | --- | --- |
| 90-100 | Excellent | Continue standard targets and document the winning patterns |
| 80-89 | Meets Standard | Continue targets and coach one improvement |
| 65-79 | Needs Coaching | Reduce new-DM target to 15 per day for three workdays and retrain the weakest area |
| Below 65 | Performance Review | Pause unsupervised outreach and retrain before resuming |
| Any serious compliance violation | Immediate Review | Pause outreach regardless of score |

## Weekly Scorecard Record

Record:

- Week starting date
- VA name
- Category scores
- Total score
- Best-performing segment
- Best-performing opener
- Primary weakness
- One improvement for next week
- Manager decision
