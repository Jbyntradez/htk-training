# HTK Outreach Operations Repository

## Purpose

This repository is the operating system for Virtual Assistants running manual X outreach for Hard To Kill Training (HTK).

The objective is simple:

1. Identify men who fit the HTK market.
2. Start relevant, human conversations.
3. Understand the prospect's goals and obstacles.
4. Confirm whether coaching is worth discussing.
5. Schedule qualified calls with Jimmy Bynum.

The VA does not sell, quote prices, diagnose injuries, prescribe training, or pressure prospects. The VA opens the right conversations and protects the brand while doing it.

## HTK Positioning

HTK is a premium performance and fitness coaching brand built by Jimmy Bynum, a former multi-sport athlete, Northern Michigan University football scholarship athlete, and United States Marine.

HTK is not bodybuilding coaching. HTK helps athletes, former athletes, military, veterans, first responders, fitness enthusiasts, and high performers develop:

- Explosiveness
- Endurance
- Mobility
- Athleticism
- Functional strength
- Durability
- Real-world performance

## Non-Negotiable Outreach Rules

- Every outreach action is performed manually by an authorized VA.
- Use one authentic HTK account. Do not create or use fake personas.
- Do not use browser scripts, bots, auto-DM tools, bulk senders, or engagement automation.
- Send only contextual messages to relevant prospects.
- Do not send identical DMs repeatedly. Use the approved script framework and adapt each message to the prospect.
- Do not send a link in the first message.
- Stop immediately when a prospect says no, asks to stop, or does not want messages.
- Never argue, shame, exaggerate, or make guarantees.
- Log every prospect and every meaningful action in the CRM.

## Daily Operating Standard

After the training ramp, the standard day is:

| KPI | Daily target | Daily cap |
| --- | ---: | ---: |
| New personalized DMs sent | 20 | 25 |
| Follow-ups sent | 15 | 20 |
| New conversations started | 6 | Not capped |
| Qualified leads created | 2 | Not capped |
| Calls scheduled | 1 | Not capped |

Active two-way replies are handled promptly and do not count toward the new-DM or follow-up caps. The technical limit on X is not the operating target. Brand quality comes first.

## Start Here: First 24 Hours

Complete these files in order:

1. Read `onboarding/VA_Handbook.md`.
2. Read `compliance/X_Safety_Rules.md`.
3. Read `compliance/Anti_Spam_Guidelines.md`.
4. Read `outreach/Prospecting_SOP.md`.
5. Read `outreach/Qualification_Framework.md`.
6. Read `outreach/Outreach_Scripts.md`.
7. Read `crm/CRM_Usage_Guide.md`.
8. Duplicate the header row from `templates/Lead_Tracker.csv` into the live tracker.
9. Complete the Day 1 tasks in `onboarding/First_Week_Training.md`.

The VA may begin live outreach only after the manager approves the first 10 draft prospect selections and first 5 drafted messages.

## Daily Workflow

Use `onboarding/Daily_Checklist.md` every workday:

1. Review the account and CRM.
2. Process active conversations first.
3. Send due follow-ups.
4. Research and log new prospects.
5. Send personalized first-contact DMs.
6. Qualify active conversations.
7. Schedule qualified calls.
8. Complete the daily report.

## Management Rhythm

- Daily: VA submits the daily report and flags any prospect requiring founder attention.
- Weekly: Manager scores performance using `management/Performance_Scorecard.md`.
- Monthly: Manager reviews targets, message quality, show rate, and lead quality before changing volume.

Do not increase outreach volume because a single week is slow. Improve prospect quality, personalization, qualification, and follow-up discipline first.
