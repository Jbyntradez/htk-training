# Anti-Spam Guidelines

## Principle

HTK does not spam.

Outreach exists to start relevant conversations with men who appear likely to value performance coaching. A prospect should be able to understand why the message was sent.

## Before Sending Any First-Contact DM

Confirm all seven:

1. The prospect fits an HTK segment.
2. The prospect is likely age 18-45 based on public context and does not appear under 18.
3. The profile has a real public signal connected to performance, training, athletic background, service, readiness, or durability.
4. The CRM contains no existing record requiring a different action.
5. The message is manually drafted and sent.
6. The message asks one useful question.
7. The message contains no link.

If any condition fails, do not send.

## Personalization Standard

A first-contact message must connect to one of:

- Public bio
- Public post
- Public reply
- Public interaction with HTK
- Clearly relevant public training context

Good:

> Saw your post about strength being solid but conditioning falling off. What kind of conditioning are you trying to get back to?

Bad:

> Hey man, saw your profile. Want to get in the best shape of your life? Book a call now.

## Frequency Rules

- Maximum 25 new personalized DMs per day.
- Maximum 20 follow-ups per day.
- Maximum three no-reply follow-ups after first contact.
- Follow-Up 1: after three calendar days.
- Follow-Up 2: five calendar days later.
- Follow-Up 3: seven calendar days later.
- Stop sooner if the prospect replies, declines, or opts out.
- Do not restart a closed sequence without manager approval or a new prospect-initiated interaction.

## Duplicate-Message Rules

X warns that duplicate DMs to multiple accounts can be treated as spam.

Therefore:

- Do not paste the same exact first-contact message repeatedly.
- Use varied approved openings.
- Adapt the message to the prospect's real context.
- Do not send repeated link-only messages.
- Do not use a bulk sender.
- Do not send a script to every follower, every liker, or every search result.

## Public Engagement Rules

Public replies can be useful when they contribute to a conversation.

Do:

- Reply when HTK has a relevant human contribution.
- Keep the reply connected to the original post.
- Respect the conversation.

Do not:

- Reply with irrelevant promotion.
- Repeat the same reply across many posts.
- Use trending hashtags to force attention toward HTK.
- Coordinate likes, follows, replies, or reposts to inflate metrics.

## Opt-Out Rules

Treat any of these as an opt-out:

- `Stop`
- `Do not message me`
- `Not interested`
- `No thanks`
- A clear request to leave the person alone

Reply once:

> Understood. You will not receive another message from us.

Then:

1. Mark `Closed Lost`.
2. Set `Opt_Out` to `Yes`.
3. Record reason `Opt-out`.
4. Do not contact again.

## Manager Audit

Every week, the manager reviews:

- 10 first-contact DMs
- Five follow-up sequences
- Every opt-out
- Any negative reply
- Any X warning or limitation
- CRM records for touched prospects

If message quality drops, reduce volume and retrain before sending more outreach.
