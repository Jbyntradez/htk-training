# HTK X Outreach Weekly Review

## Review Record

Record:

- Week starting date in `YYYY-MM-DD` format
- VA full name
- Manager full name
- Review date in `YYYY-MM-DD` format

## Compliance Gate

Answer `Yes` or `No`:

- Were all messages sent manually?
- Was only the authorized HTK account used?
- Were all opt-outs honored immediately?
- Were all first-contact messages free of links?
- Were duplicate bulk messages avoided?
- Were there zero X warnings or limitations?
- Was HTK represented accurately?

If any answer is `No`, record the incident, stop outreach when appropriate, and document the manager decision before continuing.

## Message QA

Review 10 first-contact DMs.

For each message, score:

- Prospect clearly fits HTK.
- Message references a real public signal or relevant context.
- Message is short.
- Message asks one easy-to-answer question.
- Message avoids a link, hype, and pressure.
- Message sounds distinct from the other sampled messages.

Record:

- Messages sampled
- Messages passed
- Pass rate
- Strongest opener
- Weakest opener
- Required script correction

## CRM QA

Review 10 lead records.

Check:

- Correct handle and profile URL
- Correct stage
- Segment recorded
- Public signal recorded
- Last-contact date recorded
- Next action and date recorded
- Opt-out status recorded when relevant
- Booking details complete when relevant

Record:

- Records sampled
- Complete records
- CRM completeness rate
- Required corrections

## Pipeline Review

Record:

- Qualified leads without scheduled calls
- Calls scheduled for the next seven days
- No-shows needing follow-up
- Active conversations without a next action
- Three most common closed-lost reasons
