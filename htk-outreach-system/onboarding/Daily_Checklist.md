# Daily VA Checklist

## Start Of Shift: 15 Minutes

- [ ] Confirm access to X and the CRM.
- [ ] Check for X warnings, restrictions, or unusual login activity.
- [ ] Review unread DMs.
- [ ] Review all leads with a next-action date of today or earlier.
- [ ] Review scheduled calls for the next 48 hours.
- [ ] Flag any security issue or account warning before sending messages.

## Priority 1: Active Conversations

- [ ] Reply to every active prospect conversation before prospecting.
- [ ] Ask only one qualification question at a time.
- [ ] Update the CRM after each meaningful reply.
- [ ] Move each lead to the correct stage.
- [ ] Escalate any medical, customer-service, partnership, public-figure, or brand-risk issue.

## Priority 2: Due Follow-Ups

- [ ] Send follow-ups only when the CRM shows they are due.
- [ ] Use the correct follow-up step from `outreach/Follow_Up_Scripts.md`.
- [ ] Personalize the message when context allows.
- [ ] Stop after the final follow-up.
- [ ] Stop immediately after any opt-out or rejection.
- [ ] Update the last-contact date and next action.

## Priority 3: New Prospecting

- [ ] Research prospects using `outreach/Prospecting_SOP.md`.
- [ ] Confirm each prospect fits the target market.
- [ ] Record a real public signal.
- [ ] Check the CRM for duplicates before sending anything.
- [ ] Skip minors, private accounts with no usable context, obvious spam accounts, and poor-fit profiles.
- [ ] Log each prospect as `New Lead`.

## Priority 4: First-Contact DMs

- [ ] Send a maximum of 25 new DMs.
- [ ] Aim for the daily target of 20 quality messages.
- [ ] Adapt an approved opening pattern to the prospect.
- [ ] Keep the first message short.
- [ ] Ask one easy-to-answer question.
- [ ] Do not send a link.
- [ ] Move the CRM stage to `Contacted`.
- [ ] Set Follow-Up 1 for three calendar days later.

## Priority 5: Qualification And Booking

- [ ] Confirm the prospect's performance goal.
- [ ] Confirm the main obstacle.
- [ ] Confirm why progress matters now.
- [ ] Confirm the prospect is open to coaching support.
- [ ] Confirm willingness to speak with Jimmy within 14 days.
- [ ] Move qualified prospects to `Qualified`.
- [ ] Use the manual scheduling workflow in `outreach/Booking_Process.md`.
- [ ] Move the lead to `Call Scheduled` only after the date, time, time zone, and calendar email are confirmed.

## End Of Shift: 20 Minutes

- [ ] Confirm every touched lead has a stage, last-contact date, next action, and next-action date.
- [ ] Confirm every scheduled call has complete details.
- [ ] Count daily KPIs.
- [ ] Complete `management/Daily_Report_Template.md`.
- [ ] Note the three best conversations and any blocked conversations.
- [ ] Send the report to the manager.
- [ ] Flag any prospect waiting on Jimmy or the manager.

## Stop-Work Triggers

Stop sending new outreach and alert the manager immediately if:

- X displays a warning, DM limitation, or account restriction.
- You suspect unauthorized access.
- The account receives multiple negative replies in a short period.
- You accidentally send a wrong message, wrong link, or private information.
- You are asked to use an unapproved tool or access method.
