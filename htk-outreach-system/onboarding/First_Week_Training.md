# First Week VA Training Plan

## Goal

By the end of Day 5, the VA can independently research prospects, write contextual DMs, qualify leads, schedule calls, update the CRM, and report performance while protecting the HTK brand.

The VA does not begin unsupervised live outreach until the manager approves the certification checklist.

## Day 1: Brand And Compliance

### Read

1. `README.md`
2. `onboarding/VA_Handbook.md`
3. `compliance/X_Safety_Rules.md`
4. `compliance/Anti_Spam_Guidelines.md`
5. `compliance/Brand_Protection_Policy.md`

### Complete

1. Write a five-sentence summary of what HTK is.
2. Write a five-sentence summary of what the VA is allowed to do.
3. Write a five-sentence summary of what the VA must never do.
4. Review 20 X profiles and mark each as `Fit`, `Possible Fit`, or `Do Not Contact`.
5. Explain the decision for every profile in one sentence.

### Manager Review

The manager approves Day 1 only when:

- At least 18 of 20 prospect-fit decisions are correct.
- The VA can explain why HTK is not bodybuilding coaching.
- The VA can explain why manual, contextual outreach is required.

## Day 2: Prospecting And CRM

### Read

1. `outreach/Prospecting_SOP.md`
2. `crm/Lead_Status_Definitions.md`
3. `crm/CRM_Usage_Guide.md`
4. `crm/Tracking_Workflow.md`

### Complete

1. Research 30 prospects.
2. Log 20 qualified prospecting candidates as `New Lead`.
3. Record a real public signal for every logged prospect.
4. Set a clear next action for every record.
5. Remove any prospect who meets a disqualification rule.

### Manager Review

The manager samples 10 records. Day 2 passes only when:

- Every sampled profile fits the target market.
- Every sampled record has a usable personalization note.
- Every sampled record has the correct stage and next action.
- No sensitive personal information is copied into the CRM.

## Day 3: Script Practice

### Read

1. `outreach/Outreach_Scripts.md`
2. `outreach/Follow_Up_Scripts.md`
3. `outreach/Objection_Handling.md`

### Complete

1. Draft 15 first-contact messages for manager review.
2. Use at least five different approved opening patterns.
3. Draft responses for five qualification scenarios.
4. Draft responses for five objection scenarios.
5. Revise any message that feels copied, generic, long, or sales-heavy.

### Manager Review

The manager approves the VA for supervised live outreach only when:

- Every draft is relevant to the profile.
- Every first-contact message is short and easy to answer.
- No first-contact message contains a link.
- No draft makes a promise or pressures the prospect.

## Day 4: Supervised Live Outreach

### Daily Targets

| Activity | Target | Cap |
| --- | ---: | ---: |
| New personalized DMs | 10 | 10 |
| Follow-ups | 0 | 0 |
| New conversations started | 3 | None |
| Qualified leads | 1 | None |
| Calls scheduled | 0-1 | None |

### Complete

1. Send 10 manager-approved first-contact DMs.
2. Update the CRM immediately after each send.
3. Route all responses through the qualification workflow.
4. Ask the manager to review any uncertain reply before sending it.
5. Submit the daily report.

### Manager Review

The manager reviews every sent DM and every active conversation.

## Day 5: Controlled Production

### Daily Targets

| Activity | Target | Cap |
| --- | ---: | ---: |
| New personalized DMs | 15 | 15 |
| Follow-ups | Up to 5 due messages | 5 |
| New conversations started | 4 | None |
| Qualified leads | 1 | None |
| Calls scheduled | 0-1 | None |

### Complete

1. Process active conversations first.
2. Send due follow-ups.
3. Research and log new prospects.
4. Send first-contact DMs.
5. Qualify prospects and schedule any approved call.
6. Submit the daily report.

## Certification Checklist

The manager checks each item before the VA advances to the full production target:

- Can state HTK positioning accurately.
- Can identify a target-market prospect.
- Can identify a prospect who should not be contacted.
- Can write a personalized first-contact DM in under three minutes.
- Can follow the qualification framework without interrogating.
- Can schedule a call using the manual booking workflow.
- Can log every required CRM field accurately.
- Can calculate daily KPIs.
- Can explain all stop-contact rules.
- Can identify situations requiring escalation.

## Ramp After Certification

| Workday | New DM target | Follow-up target |
| --- | ---: | ---: |
| Day 6 | 15 | 8 |
| Day 7 | 18 | 10 |
| Day 8 onward | 20 | 15 |

The steady-state caps remain 25 new DMs and 20 follow-ups per day. The manager can reduce volume at any time if quality drops or the account receives warnings.
