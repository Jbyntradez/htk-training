# 25. Quality Assurance Standards

## Objective

Ensure every coach delivers the HTK experience consistently.

## QA Review Cadence

Manager reviews:

- 5 client CRM records per coach per week.
- 5 coach messages per coach per week.
- 3 check-in responses per coach per week.
- Every injury escalation.
- Every non-compliance escalation.
- Every renewal-risk client.
- Every refund/customer complaint.

## QA Categories

1. Positioning accuracy
2. Program design quality
3. Communication quality
4. Check-in quality
5. Accountability handling
6. Scope compliance
7. CRM accuracy
8. Retention management

## Program QA Checklist

- [ ] Goal is clear.
- [ ] Program matches goal.
- [ ] Weekly structure matches client schedule.
- [ ] Strength, conditioning, mobility, and performance work are addressed appropriately.
- [ ] Progression is defined.
- [ ] Exercise selection is appropriate.
- [ ] Safety flags are addressed.
- [ ] Program changes have rationale.

## Communication QA Checklist

- [ ] Response is clear.
- [ ] Response is specific to client.
- [ ] Tone is professional.
- [ ] Coach gives actionable next step.
- [ ] Coach stays within scope.
- [ ] No guarantee, shame, or medical advice.
- [ ] Meaningful action is logged.

## Check-In QA Checklist

- [ ] Coach reviewed adherence.
- [ ] Coach reviewed obstacle.
- [ ] Coach reviewed recovery.
- [ ] Coach identified safety flags.
- [ ] Coach gave next focus.
- [ ] Coach adjusted program if needed.
- [ ] CRM updated.

## Automatic QA Failures

- Medical advice given.
- Injury ignored.
- Client told to train through concerning pain.
- Guarantee made.
- Bodybuilding-only positioning used as HTK promise.
- Client complaint ignored.
- CRM missing safety escalation.
- Inappropriate client communication.

## QA Scorecard

| Area | Points |
|---|---:|
| Goal/program alignment | 20 |
| Communication quality | 15 |
| Check-in quality | 15 |
| Accountability handling | 15 |
| Progress tracking | 10 |
| CRM accuracy | 15 |
| Brand/scope compliance | 10 |

Score interpretation:

- 90-100: Excellent
- 80-89: Meets standard
- 65-79: Needs coaching
- Below 65: Performance review
- Any automatic failure: Immediate review

## Corrective Action

1. Identify failed standard.
2. Review examples with coach.
3. Define one behavior to fix.
4. Reduce client load if quality risk exists.
5. Re-score after one week.
6. Escalate if issue repeats.
