# 24. Coach KPIs

## Objective

Measure coaching quality, client execution, retention risk, and operational reliability.

## Daily KPIs

| KPI | Target |
|---|---:|
| Check-ins reviewed | 100% due within 1 business day |
| Client messages handled | 100% due within 1 business day |
| Injury/safety escalations | 100% same business day |
| CRM updates completed | 98%+ |
| Program changes documented | 100% |
| Missed check-ins actioned | 100% same day identified |

## Weekly KPIs

| KPI | Healthy Standard |
|---|---:|
| Active client check-in completion | 85%+ |
| Coach response timeliness | 95%+ on time |
| Program review completion | 100% due reviews |
| Non-compliance interventions | 100% documented |
| Retention-risk clients tagged | 100% |
| Injury escalations handled correctly | 100% |
| CRM completeness | 98%+ |

## Monthly KPIs

| KPI | Use |
|---|---|
| Client adherence average | Execution health |
| Client retention rate | Coaching/business health |
| Renewal rate | Value and progress clarity |
| Client satisfaction score | Experience quality |
| Progress review completion | Coaching discipline |
| Escalation volume | Safety and service monitoring |
| Testimonial candidates | Success story pipeline |
| Coach QA score | Manager performance review |

## Quality KPIs

### Coaching Quality

Sampled clients should show:

- Clear goal.
- Current program block.
- Weekly feedback.
- Progress tracking.
- CRM notes.
- No ignored safety flags.

Target: 90%+ pass.

### Communication Quality

Sampled messages should be:

- Clear
- Specific
- Within scope
- Professional
- Actionable

Target: 95%+ pass.

### Retention Quality

Renewal-window clients should have:

- Progress summary
- Next goal
- Risk status
- Renewal recommendation

Target: 100%.

## KPI Diagnosis

| Pattern | Likely Issue | Management Action |
|---|---|---|
| Low check-in completion | Weak accountability | Review missed check-in protocol |
| Slow responses | Coach capacity issue | Review workload and workflow |
| High non-compliance | Poor onboarding or unrealistic plans | Audit first 30 days |
| Low retention | Progress not visible or goal mismatch | Audit progress reviews |
| High injury escalations | Programming mismatch or client population issue | Audit program design |
| Poor CRM accuracy | Rushed operations | Reduce load or retrain |
