# 15. Progress Tracking System

## Objective

Track the right data so coaching decisions are based on evidence, not guesses.

## Progress Tracking Categories

1. Adherence
2. Performance
3. Conditioning
4. Mobility
5. Recovery
6. Client feedback
7. Body metrics when approved and relevant
8. Goal-specific markers

## Required Weekly Tracking

Every active client must track:

- Sessions completed
- Sessions missed
- Main training wins
- Main obstacle
- Recovery quality
- Pain/injury flag yes/no
- Coach next action

## Goal-Specific Tracking

### Strength / Functional Strength

Track:

- Main lift or pattern progress
- Load, reps, sets, RPE
- Technique notes
- Exercise consistency

### Endurance / Conditioning

Track:

- Session completion
- Pace, distance, intervals, heart rate, or RPE when relevant
- Recovery between efforts
- Work capacity notes

### Mobility

Track:

- Limiting positions
- Drill adherence
- Movement-quality notes
- Client-reported restriction

### Explosiveness / Athleticism

Track:

- Jump/throw/sprint exposure
- Quality notes
- Landing/control notes
- Output if measured

### Durability / Real-World Performance

Track:

- Training consistency
- Recovery quality
- Readiness for work/sport/life demands
- Reduction in missed sessions due to stiffness/fatigue

## Progress Review Workflow

1. Review last 4 weeks.
2. Compare adherence to plan.
3. Compare performance markers.
4. Review subjective client feedback.
5. Identify limiting factor.
6. Decide next block emphasis.
7. Document progress summary.
8. Communicate next focus to client.

## Progress Decision Rules

If adherence is low:

- Do not assume programming failed.
- Solve schedule, accountability, or complexity first.

If adherence is high and progress is low:

- Review volume, intensity, recovery, exercise selection, and goal match.

If recovery is poor:

- Adjust load, volume, conditioning, or schedule.

If pain/injury appears:

- Follow injury escalation policy.

## Progress Summary Template

Use:

> Over the last four weeks, you completed [X/Y] planned sessions. The biggest improvement is [marker]. The main limiter is [limiter]. For the next block, we are focusing on [focus] because it supports [goal].

## CRM Rule

Every 4-week review must be logged with:

- Date
- Adherence summary
- Progress marker
- Limiting factor
- Program decision
- Client communication sent
