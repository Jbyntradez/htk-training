# 16. Client Accountability Framework

## Objective

Keep clients executing without shame, hype, or vague motivation.

Accountability means clear standards, direct feedback, and documented follow-through.

## Accountability Levels

### Level 1: Standard Accountability

Use when client is compliant.

Coach actions:

- Reinforce good execution.
- Clarify next priority.
- Keep standards visible.

### Level 2: Early Drift

Use when client misses 1-2 sessions or gives vague check-ins.

Coach actions:

- Identify the obstacle.
- Reduce complexity if needed.
- Set one non-negotiable action for next week.

### Level 3: Pattern Of Non-Execution

Use when client misses repeated sessions, skips check-ins, or does not communicate.

Coach actions:

- Name the pattern.
- Ask for a decision.
- Adjust plan to minimum effective version if appropriate.
- Document in CRM.

### Level 4: Non-Compliant Review

Use when client continues not executing after direct accountability.

Coach actions:

- Follow Non-Compliant Client Protocol.
- Escalate to manager if needed.
- Prepare retention risk note.

## Accountability Conversation Framework

1. State the observed pattern.
2. Connect it to the goal.
3. Ask what is blocking execution.
4. Set one clear action.
5. Confirm client agreement.
6. Document in CRM.

Example:

> You missed two of the three planned sessions this week. The issue is not one bad week; the issue is that your goal depends on consistency. What blocked execution, and what needs to change next week?

## Minimum Effective Week

Use when life is busy but client still wants progress.

Coach defines:

- Minimum sessions
- Shortened session option
- Non-negotiable mobility or conditioning dose
- Check-in requirement

Rule:

Minimum effective work is better than disappearing.

## Accountability Decision Rules

Support when:

- Client communicates honestly.
- Client has real constraints.
- Client wants to adjust and continue.

Challenge when:

- Client avoids check-ins.
- Client repeats excuses without action.
- Client wants results without execution.

Escalate when:

- Client is non-compliant for multiple weeks.
- Client becomes hostile.
- Client blames coach while not executing.
- Client wants refund, pause, or exception.
