# 18. Non-Compliant Client Protocol

## Objective

Address repeated non-execution before it damages results, retention, and brand trust.

## Non-Compliance Definition

A client is non-compliant when they repeatedly:

- Miss sessions.
- Miss check-ins.
- Ignore coach feedback.
- Do not communicate.
- Refuse agreed actions.
- Ask for changes while not executing.
- Blame the program without completing it.

## Non-Compliance Is Not

- One hard week.
- Illness.
- Approved travel.
- Family emergency.
- A client honestly asking for adjustment.

## Protocol Workflow

1. Identify the pattern.
2. Confirm it is not an approved exception.
3. Send direct accountability message.
4. Offer adjustment if the plan is unrealistic.
5. Set one-week reset target.
6. Document in CRM.
7. Review next week.
8. Escalate if pattern continues.

## Accountability Reset Message

Use:

> We need to reset. The goal you gave me requires consistent execution, and right now the pattern is [missed sessions/check-ins/no communication]. If the plan is too much, we can adjust it. But if you want the result, disappearing cannot be the standard. What are you willing to commit to this week?

## One-Week Reset

Coach sets:

- Minimum training requirement
- Check-in requirement
- Communication requirement
- Follow-up date

Example:

> This week the minimum is two sessions and one check-in by Friday. If that is not realistic, tell me now so we can solve the actual constraint.

## Escalate If

- Client misses reset target.
- Client refuses accountability.
- Client becomes hostile.
- Client requests refund/pause.
- Client says coaching is failing while not executing.
- Client has safety or medical issue.

## Manager Review Packet

Include:

- Client name
- Goal
- Start date
- Non-compliance pattern
- Coach messages sent
- Program adjustments offered
- Client response
- Recommended action

## Possible Manager Actions

- Continue with accountability plan.
- Reduce plan complexity.
- Pause and reset expectations.
- Move to retention conversation.
- Close client relationship if appropriate.
