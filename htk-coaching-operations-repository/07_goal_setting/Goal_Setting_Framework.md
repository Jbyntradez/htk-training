# 7. Goal Setting Framework

## Objective

Convert the client's stated desire into a performance-focused coaching target with clear execution standards.

## Goal Categories

Every client must have one primary goal category:

- Explosiveness
- Endurance
- Mobility
- Athleticism
- Functional strength
- Durability
- Real-world performance
- Return to sport or training
- Tactical/job readiness
- Consistency and structure

## Goal Setting Workflow

1. Capture the client's stated goal.
2. Translate it into performance language.
3. Identify why it matters.
4. Identify the main constraint.
5. Define a measurable success marker.
6. Define weekly execution targets.
7. Define first 30-day focus.
8. Confirm the client understands the goal.
9. Record in CRM.

## Goal Translation Examples

| Client Says | HTK Translation |
|---|---|
| I want to get athletic again | Improve movement quality, conditioning, power exposure, and strength balance |
| I am strong but out of shape | Maintain strength while building aerobic/conditioning capacity |
| I feel stiff | Improve mobility and usable positions for training |
| I need to pass a fitness test | Build test-specific readiness, conditioning, and movement capacity |
| I want to stop falling off | Build sustainable schedule, accountability, and minimum effective training structure |

## Good Goal Standard

A good goal is:

- Performance-based
- Clear
- Relevant to the client
- Connected to a reason
- Measurable enough to track
- Supported by weekly actions
- Realistic within the client's constraints

## Goal Checklist

- [ ] Primary goal category selected.
- [ ] Client's words captured.
- [ ] Coach translation documented.
- [ ] Why-now reason documented.
- [ ] Main obstacle documented.
- [ ] 30-day focus documented.
- [ ] 90-day target documented.
- [ ] Weekly execution standard documented.

## First 30-Day Focus

The first 30 days usually prioritize:

- Consistency
- Technique
- Baseline conditioning
- Mobility exposure
- Training rhythm
- Recovery awareness
- Communication habits

Do not overload a new client with every possible goal at once.

## Goal Review Cadence

| Timing | Review |
|---|---|
| Week 1 | Confirm client understands plan and execution standard |
| Week 2 | Check adherence and early friction |
| Week 4 | Review baseline response and adjust focus |
| Every 4 weeks | Review progress markers and next block |
| Renewal window | Tie progress to next performance target |

## Goal Change Rules

Change a goal only when:

- Client achieves the original goal.
- Client's sport/work/life demand changes.
- Assessment data shows original target is not the limiting factor.
- Client cannot execute due to documented constraints.
- Manager approves a major scope shift.

Do not change goals because the client is bored after one hard week.
