# 4. Client Lifecycle

## Lifecycle Stages

Client Onboarding -> Initial Assessment -> Goal Setting -> Program Build -> Weekly Coaching -> Progress Review -> Retention/Renewal -> Alumni/Referral

## Stage 1: Client Onboarding

Definition: Client has paid and is entering the coaching system.

Required actions:

- Confirm handoff from sales.
- Confirm onboarding instructions sent.
- Confirm intake form completion.
- Confirm client access to required platform.
- Record client profile in CRM.
- Assign coach.

Move forward when:

- Intake is complete and assessment is scheduled or submitted.

## Stage 2: Initial Assessment

Definition: Coach reviews client history, goals, constraints, movement context, training readiness, and baseline performance markers.

Required actions:

- Review intake.
- Identify goal category.
- Identify training background.
- Identify constraints.
- Identify injury/scope flags.
- Establish baseline markers.
- Escalate any safety concern.

Move forward when:

- Coach has enough information to set goals and build the first training block.

## Stage 3: Goal Setting

Definition: Coach and client align on measurable priorities.

Required actions:

- Define primary goal.
- Define secondary goal.
- Define success markers.
- Define weekly execution targets.
- Define constraints.
- Define first 30-day focus.

Move forward when:

- Goals are documented in CRM and client understands the execution standard.

## Stage 4: Program Build

Definition: Coach creates or assigns the first training block.

Required actions:

- Choose training emphasis.
- Select exercise categories.
- Set weekly structure.
- Set progression method.
- Add mobility, conditioning, and performance work.
- Document rationale.

Move forward when:

- Client receives plan and first-week instructions.

## Stage 5: Weekly Coaching

Definition: Client executes, checks in, receives feedback, and gets adjustments.

Required actions:

- Review check-ins.
- Review training notes.
- Track adherence.
- Adjust plan when needed.
- Reinforce standards.
- Escalate non-compliance or safety issues.

Move forward when:

- Client completes training block or reaches review point.

## Stage 6: Progress Review

Definition: Coach evaluates whether the plan is producing the intended adaptation.

Required actions:

- Compare baseline to current markers.
- Review adherence.
- Review subjective feedback.
- Identify limiting factor.
- Decide next block emphasis.
- Update CRM.

Move forward when:

- Next block is planned and client understands the next focus.

## Stage 7: Retention/Renewal

Definition: Client is approaching renewal, continuation, pause, or completion.

Required actions:

- Review progress.
- Identify next performance target.
- Prepare renewal recommendation.
- Escalate churn risk.
- Document decision.

Move forward when:

- Client renews, transitions, pauses, or exits with documented reason.

## Stage 8: Alumni/Referral

Definition: Client completes or leaves active coaching.

Required actions:

- Confirm final status.
- Collect approved feedback/testimonial if appropriate.
- Record referral opportunity.
- Tag alumni status.
- Set approved future follow-up if allowed.

## Lifecycle Rules

- No client enters active programming without intake review.
- No program starts if a safety flag requires escalation.
- No client stays active without a current goal and next action.
- No client reaches renewal without a progress review.
- No testimonial is collected without approval and documented consent.
