# 9. Exercise Selection Guidelines

## Objective

Select exercises that support the client's performance goal, readiness, equipment, and movement quality.

## Exercise Selection Filters

Before adding an exercise, ask:

1. What adaptation does this exercise support?
2. Does the client have the skill to perform it?
3. Does the client have the equipment?
4. Can the client recover from it?
5. Does it fit the weekly plan?
6. Is there a safer or simpler option that accomplishes the same goal?
7. Does it create unnecessary orthopedic or technical risk?

## Exercise Categories

### Strength

Possible categories:

- Squat pattern
- Hinge pattern
- Push pattern
- Pull pattern
- Lunge/split stance
- Carry
- Trunk stability
- Grip/support

### Conditioning

Possible categories:

- Aerobic base
- Tempo intervals
- Repeated sprint ability
- Mixed modal conditioning
- Loaded conditioning
- Zone-based endurance
- Sport/test-specific conditioning

### Mobility / Movement Preparation

Possible categories:

- Hip mobility
- Ankle mobility
- Thoracic mobility
- Shoulder control
- Breathing/bracing
- Dynamic warm-up
- Controlled range work

### Explosive / Athletic

Possible categories:

- Jumps
- Throws
- Bounds
- Sprints
- Acceleration drills
- Deceleration drills
- Change of direction
- Low-amplitude plyometrics

## Selection Decision Rules

### Choose Simple When

- Client is new.
- Client is inconsistent.
- Technique is unknown.
- Equipment is limited.
- Recovery is poor.
- Goal does not require complexity.

### Choose Advanced When

- Client demonstrates readiness.
- Movement quality is consistent.
- The exercise directly supports the goal.
- Recovery is sufficient.
- Coach can monitor execution through available tools.

### Avoid Or Regress When

- Client reports pain.
- Technique cannot be controlled.
- Exercise creates unnecessary risk.
- Client is using load to compensate for poor movement.
- Client cannot explain or execute the intent.

## Substitution Rules

Substitute an exercise when:

- Equipment is unavailable.
- Movement quality is poor.
- Client reports discomfort that requires modification.
- Exercise does not match goal.
- Travel/work schedule requires alternate plan.

Substitution note must include:

- Original exercise
- Replacement exercise
- Reason
- Whether change is temporary or permanent

## Exercise Quality Standard

A good HTK exercise selection is:

- Purposeful
- Coachable
- Progressable
- Appropriate to readiness
- Connected to the goal
- Not selected for novelty alone
