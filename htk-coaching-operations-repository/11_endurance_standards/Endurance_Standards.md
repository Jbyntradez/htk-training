# 11. Endurance Standards

## Objective

Build conditioning and endurance that support the client's performance demands without turning training into random punishment.

## Endurance Categories

HTK may use:

- Aerobic base work
- Tempo intervals
- Threshold work
- Repeated sprint work
- Loaded conditioning
- Mixed modal conditioning
- Sport/test-specific conditioning
- Recovery conditioning

## Conditioning Decision Rules

Choose aerobic base when:

- Client is deconditioned.
- Recovery between efforts is poor.
- General work capacity is low.
- Client needs sustainable fitness.

Choose intervals when:

- Client has a base.
- Goal requires higher output.
- Client can recover between sessions.
- Technique and readiness are stable.

Choose sport/test-specific conditioning when:

- Client has a known event, test, or readiness demand.
- Timeline and standards are clear.
- Training can progress safely.

Choose loaded conditioning carefully when:

- Client has movement competency.
- Load supports the goal.
- Fatigue will not create unacceptable technical breakdown.

## Conditioning Workflow

1. Identify conditioning goal.
2. Establish baseline.
3. Choose conditioning type.
4. Set frequency.
5. Set intensity method.
6. Set progression.
7. Track response.
8. Adjust based on recovery and performance.

## Intensity Control Options

Use one or more:

- RPE
- Pace
- Heart rate if available
- Time
- Distance
- Work/rest intervals
- Talk-test style notes
- Repeated effort quality

## Progression Standards

Progress one variable at a time:

- Duration
- Distance
- Pace
- Work interval
- Rest interval
- Rounds
- Load
- Frequency

Do not increase everything at once.

## Conditioning Check-In Questions

- "What conditioning sessions did you complete?"
- "What was the hardest part?"
- "Did conditioning affect strength sessions?"
- "How did recovery feel?"
- "Did you feel better, worse, or the same by the end of the week?"

## Red Flags

Escalate or adjust when:

- Client reports chest pain, faintness, severe shortness of breath, or concerning symptoms.
- Client cannot recover between sessions.
- Conditioning repeatedly causes missed strength work.
- Client reports pain that changes movement.
- Client is using conditioning as punishment.

## Endurance Success Markers

Possible markers:

- Lower perceived effort at same workload.
- Higher output at same RPE.
- Better recovery between sets.
- Better work capacity.
- Improved test/event performance.
- Better energy across the week.
