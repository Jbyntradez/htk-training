# 26. Coach Certification Program

## Objective

Certify coaches before they handle live HTK clients independently.

## Certification Requirements

A coach must demonstrate:

- HTK positioning
- Coaching philosophy
- Program design standards
- Initial assessment process
- Goal setting
- Mobility, endurance, and explosive standards
- Communication rules
- Check-in process
- Progress tracking
- Accountability protocols
- Injury escalation
- Scope of practice
- CRM usage
- Brand protection

## Training Sequence

### Phase 1: HTK Positioning

Read:

- `01_positioning/HTK_Positioning.md`
- `03_coaching_philosophy/HTK_Coaching_Philosophy.md`
- `27_brand_protection/HTK_Brand_Protection_Rules.md`

Task:

- Explain HTK in 60 seconds.
- Explain what HTK is not.
- Identify three prohibited claims.

### Phase 2: Client Intake and Assessment

Read:

- `05_client_onboarding/Client_Onboarding_SOP.md`
- `06_initial_assessment/Initial_Assessment_Process.md`
- `07_goal_setting/Goal_Setting_Framework.md`

Task:

- Complete two mock intake reviews.
- Identify goals, constraints, and safety flags.

### Phase 3: Program Design

Read:

- `08_program_design/Program_Design_Standards.md`
- `09_exercise_selection/Exercise_Selection_Guidelines.md`
- `10_mobility_standards/Mobility_Standards.md`
- `11_endurance_standards/Endurance_Standards.md`
- `12_explosive_performance/Explosive_Performance_Standards.md`

Task:

- Build two sample 4-week program outlines.
- Explain exercise selection and progression.

### Phase 4: Client Communication and Check-Ins

Read:

- `13_client_communication/Client_Communication_Rules.md`
- `14_weekly_checkin/Weekly_Check_In_Process.md`
- `16_accountability/Client_Accountability_Framework.md`

- Write one accountability reset message.

### Phase 5: Safety and Scope

Read:

- `19_injury_escalation/Injury_Escalation_Policy.md`
- `20_scope_of_practice/Scope_Of_Practice_Rules.md`

Task:

- Classify ten scenarios as coach, modify, escalate, or refer out.

### Phase 6: CRM and QA

Read:

- `23_crm/CRM_Requirements.md`
- `24_coach_kpis/Coach_KPIs.md`
- `25_quality_assurance/Quality_Assurance_Standards.md`

Task:

- Complete three sample CRM records.
- Pass QA scorecard review.

## Certification Exam

Coach must complete:

1. Written positioning test.
2. Intake/assessment review.
3. Program design exercise.
4. Check-in response drill.
5. Injury/scope scenario drill.
6. CRM record review.
7. Manager interview.

## Passing Standard

| Area | Points |
|---|---:|
| Positioning | 10 |
| Assessment | 15 |
| Goal setting | 10 |
| Program design | 20 |
| Communication | 15 |
| Accountability | 10 |
| CRM | 10 |
| Scope/brand compliance | Pass/fail |

Passing:

- 80+ points.
- 0 scope failures.
- 0 medical advice errors.
- 0 brand-protection failures.

## Live Client Ramp

### First 5 Clients

- Manager reviews every intake.
- Manager reviews first program block.
- Manager reviews first two check-in responses.

### Clients 6-15

- Manager samples 50%.
- Weekly QA review.

### After 15 Clients

- Standard QA cadence.
- Any serious scope/brand issue returns coach to supervised status.
